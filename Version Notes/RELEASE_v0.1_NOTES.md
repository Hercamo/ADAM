# Version 0.1

First version of ADAM supporting the book *ADAM - Autonomy Doctrine and Architecture Diagram*. Includes entire AI Specification Pack.

## What's in this release

- **ADAM - The Story Deck v0.1.pptx** — Executive presentation
- **ADAM - Support Documents/** — 11 canonical specifications (Agent Org Chart, BOSS Score Formulas, Copyright & Use Agreement, DNA Deployment Specification, DNA Questionnaire, Governance Charter, Industry Services & Architecture, Intent Object Definition, Keyword Glossary, Legal & Arbitration Sections, Appendix 9 — Comparative Landscape)
- **ADAM - Architecture Diagrams v0.1/** — 9 editable draw.io topologies (AWS, Azure, GCP, on-prem, multi-cloud, fully sovereign, open-source Kubernetes)
- **ADAM - DNA Tool v0.1/** — Conversational web app (FastAPI + React + Helm)
- **ADAM - DNA Deployment Tool v0.1/** — Python CLI + per-platform IaC generators + NetStreamX worked example
- **ADAM - AGT LIGHT Plugin v0.1/** — Reference runtime integration for Microsoft AGT
- **ADAM - AGT-Plugin - FULL AGT Implementation v0.1/** — Full Microsoft AGT integration
- **ADAM - Financial Models - NetStreamX Fictional Company/** — Financial models for two deployment topologies
- **ADAM - Graphics/** — Brand graphics

## Release asset

`ADAM-v0.1.zip` — full snapshot of the entire ADAM/ directory (excludes `.git/` and Python `__pycache__/`).
